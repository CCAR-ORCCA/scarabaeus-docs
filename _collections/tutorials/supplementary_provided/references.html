supplementary_provided — file inventory#

Files stored here are too large or mission-specific to auto-download via supp_data.py, or are hand-crafted inputs used by the SCB tutorials. They must be present locally.

orex_real_data.tm#

OSIRIS-REx SPICE metakernel listing all kernels needed for the real-data OD tutorials (2023 mission coverage). Originally orx_2023_v01.tm by Alyssa Bailey (NAIF/JPL, October 2023); adapted for Scarabaeus by Giovanni Fereoli (CU Boulder ORCCA, May 2025). All kernel paths in this file are relative to the supplementary data/ folder and are downloaded automatically by load_data() from NAIF.

JSON measurement files — format note#

All .json files below originate from the official OSIRIS-REx tracking data archive (TRK-2-34 format, PDS4 node: https://sbnarchive.psi.edu/pds4/orex/orex.radioscience_v1.0/trk234_trknav/). They were converted into a Scarabaeus-readable format using DeepWhaleNetwork, a conversion tool that wraps NASA-PDS/PyTrk234 to parse DSN TRK-2-34 binary files and export them as structured JSON. DeepWhaleNetwork is not yet publicly released; for more information contact the ORCCA Lab.

2018_152_2018_182_tro.json#

Tropospheric media-correction data for OSIRIS-REx radiometric measurements, covering 2018 day-of-year 152–182 (June 1 – July 1). Used in the real-data radiometric OD tutorial to apply atmospheric range-rate corrections.

orex_beno_2018_158_184515_2018_158_230501_65.json#

OSIRIS-REx original tracking data (TRK-2-34), converted to Scarabaeus JSON format. Coverage: 2018 DOY 158, 18:45:15 – 23:05:01 UTC | 65 observations.

orex_beno_2018_159_054533_2018_159_104001_35.json#

OSIRIS-REx original tracking data (TRK-2-34), converted to Scarabaeus JSON format. Coverage: 2018 DOY 159, 05:45:33 – 10:40:01 UTC | 35 observations.

orex_beno_2018_159_183015_2018_159_230501_65.json#

OSIRIS-REx original tracking data (TRK-2-34), converted to Scarabaeus JSON format. Coverage: 2018 DOY 159, 18:30:15 – 23:05:01 UTC | 65 observations.

example.mission_maneuver_spec#

Generic example maneuver specification file used by the SCB maneuver-modeling tutorial to illustrate the expected input format for finite-burn sequences. This file does not represent or reflect the Emirates Mars Mission in any way. Source: Jeremy Knittle (personal communication).
