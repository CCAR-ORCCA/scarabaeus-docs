DatabaseManager#

class DatabaseManager(mode='local', file='db_config.json')#

Bases: object

Interface for connecting to and querying the Scarabaeus data store.

Supports two back-ends selected at construction time:

  • "mongo" — remote MongoDB instance on the LASP server, accessed through an SSH tunnel. Credentials are read from environment variables LASP_USER_NAME and LASP_PASSWORD; non-sensitive connection parameters come from a JSON config file.

  • "local" — file-system-based store using a local config file.

Parameters:

  • mode ({"local", "mongo"}, optional) – Back-end to connect to. Defaults to "local".

  • file (str, optional) – Name of the JSON configuration file (relative to <cwd>/config/). Defaults to "db_config.json".

Notes

This class is a work-in-progress; the interface may change as additional back-ends and query capabilities are added.

get_outlier_flags(process_id=None, user_id=None) → str#

Retrieve outlier flags for all iterations belonging to a process.

If neither process_id nor user_id is given, the most recently created process is used.

Parameters:

  • process_id (str or bson.ObjectId, optional) – ID of the target process document. Strings are automatically converted to ObjectId.

  • user_id (str, optional) – Creator identifier (created_by field). Used to locate the most recent process for that user when process_id is None.

Returns:

Formatted multi-line report listing the process metadata and per-iteration outlier flag dictionaries. Returns a plain message string if no matching process is found.

Return type:

str

get_output_file(output_type: str, process_id='final', iteration_id='final')#

Retrieve and download a file from MongoDB GridFS based on process and iteration identifiers. This method queries the ‘iterations’ collection to find the document with the specified process_id and iteration_id, retrieves the file_id from the specified field_name, and downloads the file from GridFS.

Parameters:

  • output_type (str) – The type of the output. Must be one of “plot”, “covariance”, “json”, “orbiter_est”.

  • process_id (str, optional) – The ID of the process.

  • iteration_id – The ID of the iteration.

Returns:

The path to the downloaded file.

Return type:

str

Raises:

ValueError – If no document is found with the given process_id and iteration_id, or if the field_name is not found in the document.

#process_id is optional, default is “final” - queries the recently created process #iteration_id is optional, default is “final” - queries the recently created iteration
