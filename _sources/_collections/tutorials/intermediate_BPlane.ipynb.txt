    "## 3. B-Plane Object\n",
    "\n",
    "`Bplane.__init__` does three things automatically:\n",
    "\n",
    "1. **Writes a SPICE frame-kernel (`.tf`) file** — `fk_file` is a plain-text file\n",
    "   that registers a new named SPICE frame (`OREX_EARTH_BPLANE`) in the kernel pool.\n",
    "   This lets SPICE rotate any vector into the B-plane coordinate system on demand.\n",
    "   The file is written (or overwritten) every time a `Bplane` object is created with\n",
    "   `new_bplane=True`.\n",
    "\n",
    "2. **Computes and stores the reference aimpoint at construction time** — immediately\n",
    "   after loading the frame, it calls `SpiceManager.get_state` to retrieve the SPICE\n",
    "   trajectory of OSIRIS-REx relative to Earth at the TCM epoch, then runs\n",
    "   `compute_hyperbolic_parameters` on that state.  The resulting\n",
    "   (B·T\\*, B·R\\*, ltof\\*) triple is stored internally as `_bt_ref`, `_br_ref`,\n",
    "   `_ltof_ref`.  **No separate \"set target\" step is needed** — every call to\n",
    "   `linear_targeting` or `nonlinear_targeting` will automatically aim for these values.\n"

    "## 9. Covariance Mapping\n",
    "\n",
    "`target_covariance` maps the 6×6 state covariance **P**_sv into the 3×3\n",
    "B-plane covariance **P**_B via the B-plane Jacobian **M**:\n",
    "\n",
    "$$\\mathbf{P}_B = \\mathbf{M}\\,\\mathbf{P}_{sv}\\,\\mathbf{M}^\\top,\n",
    "\\qquad\n",
    "\\mathbf{M} = \\frac{\\partial[B{\\cdot}T,\\;B{\\cdot}R,\\;\\ell_{\\rm tof}]}{\\partial[\\mathbf{r},\\mathbf{v}]}\n",
    "\\in \\mathbb{R}^{3\\times 6}$$\n",
    "\n",
    "**This is a static, linear projection** — it maps the OD covariance from the\n",
    "TCM epoch (30 days before closest approach) directly onto the B-plane *without*\n",
    "numerically propagating uncertainties forward in time.  No STM integration, no\n",
    "Monte Carlo.  The Jacobian **M** is evaluated once at the current (perturbed)\n",
    "state via finite differences of `compute_hyperbolic_parameters`.\n",
    "\n",
    "The approximation is accurate whenever the B-plane mapping is locally linear\n",
    "over the uncertainty cloud, which holds for well-converged OD solutions with\n",
    "position uncertainty well below the B-plane gradient scale (~hundreds of km for\n",
    "a typical Earth flyby)."
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,

   "outputs": [],

    "sv_cov = np.diag([\n",
    "    pos_1sigma**2, pos_1sigma**2, pos_1sigma**2,\n",
    "    vel_1sigma**2, vel_1sigma**2, vel_1sigma**2,\n",
    "])\n",
    "\n",
    "P_bplane = BplaneObj.target_covariance(sv_cov, sc_rel_state_j2000=state_pert_tcm)\n",
    "\n",
    "sigma_bt   = np.sqrt(float(P_bplane.values[0, 0]))\n",
    "sigma_br   = np.sqrt(float(P_bplane.values[1, 1]))\n",
    "sigma_ltof = np.sqrt(float(P_bplane.values[2, 2]))\n",
    "rho_btbr   = float(P_bplane.values[0, 1]) / (sigma_bt * sigma_br)\n",
    "\n",
    "print(\"─\" * 55)\n",
    "print(\"  1-σ B-plane uncertainties (mapped from OD covariance)\")\n",

    "print(f\"  σ(B·R)  =  {sigma_br:.4f}  km\")\n",
    "print(f\"  σ(ltof) =  {sigma_ltof:.4f}  s\")\n",
