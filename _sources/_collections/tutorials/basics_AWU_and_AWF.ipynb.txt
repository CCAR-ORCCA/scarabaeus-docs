    "et0    = scb.SpiceManager.jd2et(2461809.72995654)   # arbitrary JD\n",
    "epoch  = scb.EpochArray(np.array([et0]), sys='TDB')\n",

    "# ── Direction Cosine Matrix: J2000 → IAUEARTH ────────────────────\n",
    "# get_DCM returns a plain numpy ndarray (shape 3×3)\n",

    "## Summary\n",
    "\n",
    "| Concept | Class | Key methods |\n",
    "|---------|-------|-------------|\n",
    "| Physical units | `Units` | `get_units()`, arithmetic (`/`, `*`, `**`) |\n",
    "| Array + units | `ArrayWUnits` | `.values`, `.units`, `.convert_to()`, `+−×÷` |\n",
    "| Coordinate frame | `Frame` | `generate_common_frames()`, `get_DCM()`, `get_transformation()` |\n",
    "| Array + units + frame | `ArrayWFrame` | `.quantity`, `.frame`, `.convert_to(frame, epoch)`, `+−` |\n",
    "\n",
    "**Key safety guarantees**\n",
    "\n",
    "* `ArrayWUnits` blocks addition/subtraction between incompatible dimensions (km + km/s raises `ValueError`)\n",
    "* `ArrayWFrame` blocks arithmetic between different frames (J2000 + ITRF93 raises an exception)\n",
    "* Frame conversions require an epoch because the rotation matrix is time-varying\n",
    "\n",
    "These guarantees propagate through all Scarabaeus dynamics, filter, and measurement modules."
