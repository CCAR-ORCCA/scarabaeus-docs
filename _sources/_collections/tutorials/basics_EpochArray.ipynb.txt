    "## Creating EpochArrays\n",
    "EpochArray utilizes the SPICE time subsystem to perform conversions and provide representations. For a thorough explanation of SPICE's time subsystem, see [SPICE Time Subsystem Required Reading](https://naif.jpl.nasa.gov/pub/naif/toolkit_docs/C/req/time.html).\n",
    "\n",
    "All of these different time systems and representations can sometimes be cumbersome to track while working on a script. EpochArray is intended to alleviate some of this overhead and make working with times as simple as possible. The following section will cover the most common ways to construct an EpochArray, but for an exhaustive list, see the EpochArray documentation.\n",
    "\n",
    "### EpochArray Error Handling\n",
    "EpochArray implements a robust set of error handling to make clear why something may be failing. This section will provide examples that raise a few of these errors to illustrate common pitfalls when working with time in SCB\n",
    "\n",
    "- **NOTE**: If an EpochArray error is raised as the result of another error deeper within SPICE, the SPICE error will also be noted at the end, e.g. `-- SPICE(UNPARSEDTIME)`.\n",
    "\n",
    "---\n",
    "#### MissingLSKError\n",
    "All times in EpochArray are internally defined in Barycentric Dynamical Time (TDB), also known as Ephemeris Time (ET). This maintains consistency when performing time operations, but it also requires that a leapsecond kernel (LSK) be loaded if attempting to use to any non-uniform time representations like ISOC or CAL."
   ]
  },
  {

   "outputs": [],
   "source": [
    "# make sure the kernel pool is clean\n",
    "scb.SpiceManager.clear_kernels()\n",
    "\n",
    "# uniform systems work without an LSK as long as the format is also uniform\n",
    "eparr_tdb = scb.EpochArray(0.0)  # default system is TDB, default rep is NUM\n",
    "print(f'** Uniform time systems/representations work without an LSK **\\n{eparr_tdb}\\n')\n",
    "\n",
    "# throws error if missing leapseconds when providing non-uniform definition\n",
    "try:\n",
    "    scb.EpochArray('2026 JUN 01')\n",
    "except Exception as e:\n",
    "    print('** Providing a non-uniform time without a loaded leapsecond '\n",
    "          f'kernel will raise an error **\\n'\n",
    "          f'{type(e).__name__}: {e}\\n')\n",
    "    \n",
    "# or attempting to set a non-uniform representation even if the definiton is uniform\n",
    "try:\n",
    "    scb.EpochArray(0.0, rep = 'ISOC')\n",
    "except Exception as e:\n",
    "    print('** Requesting a non-uniform time representation without a loaded '\n",
    "          f'leapsecond kernel will raise an error **\\n'\n",
    "          f'{type(e).__name__}: {e}\\n')"

    "### Array Functionality\n",
    "EpochArray fully supports scalar and array-like definitions:"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "id": "14",

    "# use the same epoch for simplicity\n",
    "time_str   = '2026 JUN 01 00:00:00.000'\n",
    "\n",
    "# define and print different definition shapes\n",
    "eparr_sclr = scb.EpochArray(time_str)                         # scalar\n",
    "eparr_vec  = scb.EpochArray([time_str, time_str, time_str])   # vector (1D array)\n",
    "eparr_mat  = scb.EpochArray(np.array([[time_str, time_str],   # matrix (nD array)\n",
    "                                        [time_str, time_str]]))\n",
    "\n",
    "for eparr in [eparr_sclr, eparr_vec, eparr_mat]:\n",
    "    print(eparr)"

    "Additionally, instead of manually creating every entry in a vector of times, use ``EpochArray.interval()``. This function supports two different interval construction methods, one that defines the interval using a given time step `dt`, and the other where it defines the interval using a given number of total epochs `n_epochs`. Both are shown below:"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "id": "16",
   "metadata": {},
   "outputs": [],
   "source": [
    "# two day interval with points every 12 hours\n",
    "epochs = scb.EpochArray.interval(start = '2026 JUN 01', end = '2026 JUN 03',\n",
    "                                   dt = scb.ArrayWUnits(12, hr),\n",
    "                                   sys = 'TDB', rep = 'CAL')\n",
    "dur = epochs.duration(units = day)  # can also get the duration sequential times\n",
    "print(f'** Step-defined epoch interval with duration {dur}:\\n{epochs}')\n",
    "\n",
    "# two day interval with 3 total epochs\n",
    "epochs = scb.EpochArray.interval(start = '2026 JUN 01', end = '2026 JUN 03',\n",
    "                                   n_epochs = 3, sys = 'TT', rep = 'CAL')\n",
    "dur = epochs.duration(units = day)\n",
    "print(f'\\n** Count-defined epoch interval with duration {dur}:\\n{epochs}')"
   ]
  },

    "## EpochArray Operations\n",
    "Can only add time to an Epocharray. Can only subtract time or another EpochArray from an EpochArray:"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "id": "19",
   "metadata": {},
   "outputs": [],
   "source": [
    "a = scb.EpochArray('2026 JUN 01', rep = 'JDTDB')\n",
    "b = scb.EpochArray('2026 JUN 08', rep = 'JDTDB')\n",
    "\n",
    "# add time to an epoch\n",
    "a_plus_1_day = a + scb.ArrayWUnits(1, day)\n",
    "print(f'a = {a}')\n",
    "print(f'a + 1 day = {a_plus_1_day}\\n')\n",
    "\n",
    "# subtract time from an epoch\n",
    "a_minus_1_day = a - scb.ArrayWUnits(1, day)\n",
    "print(f'a = {a}')\n",
    "print(f'a - 1 day = {a_minus_1_day}\\n')\n",
    "\n",
    "# subtract an epoch from an epoch to get a time\n",
    "b_minus_a = b - a\n",
    "print(f'a = {a}, b = {b}')\n",
    "print(f'b - a = {b_minus_a.convert_to(day)}')"
   ]

    "## Converting Between EpochArrays\n",
    "Can perform conersions between systems and representaitons using `.to()`:"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
