    "|:---:|:---:|---|\n",
    "| 0 | `EVENTNAME` | Segment identifier |\n",
    "| 1 | `NUMBER_OF_MANEUVERS` | Maneuvers per row (always `1` here) |\n",

    "## 4. Build Force Models and Propagators\n",
    "\n",
    "Each arc gets its own `ForceModelTranslation` and `Propagator`.\n",
    "\n",
    "| Arc | `finite_burn` | `maneuver` |\n",
    "|---|---|---|\n",
    "| Coast (before burn) | `False` | — |\n",
    "| Finite burn arc | `True` | `burns[0]` |\n",
    "| Final coast (after burn) | `False` | — |\n",
    "\n",
    "The integration step is `dt = 3600 s` (1 hour).\n",
    "\n",
    "**Non-first arcs** are initialised with `StateDefinition.empty_pv`:\n",
    "\n",
    "```python\n",
    "sv = StateArray(\n",
    "    epoch=EpochArray(np.zeros(1), sys='TDB'),\n",
    "    origin=origin,\n",
    "    state=StateDefinition.empty_pv(Orbiter, frame=J2000),\n",
    ")\n",
    "```\n",
    "\n",
    "`empty_pv` allocates a NaN position/velocity in `km` / `km/s` and marks\n",
    "them `estimated` + `dynamic`.  The epoch zero is a placeholder — the\n",
    "sequence propagator replaces both the NaN state and the epoch with the\n",
    "terminal state of the preceding arc before integration begins."
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "id": "8",

   "outputs": [],
   "source": [
    "dt = scb.ArrayWUnits(1, hr)\n",
    "dt_sec = 3600.0 \n",
    "\n",
    "def empty_state(spacecraft, origin, frame):\n",
    "    \"\"\"StateArray for non-first legs using StateDefinition.empty_pv.\n",
    "\n",
    "    StateDefinition.empty_pv fills position and velocity with NaN\n",
    "    (units km / km/s by default) and marks them as 'estimated' / 'dynamic'.\n",
    "    The epoch is set to zero — the sequence propagator will overwrite both\n",
    "    the NaN values and the epoch at run-time from the preceding arc's\n",
    "    terminal state.\n",
    "    \"\"\"\n",
    "    return scb.StateArray(\n",
    "        epoch=scb.EpochArray(np.zeros(1), sys='TDB'),\n",
    "        origin=origin,\n",
    "        state=scb.StateDefinition.empty_pv(spacecraft, frame=frame),\n",
    "    )\n",
    "\n",
    "# ── Coast before burn 0 (1 day) ──────────────────────────────────\n",
    "ep_coast0   = scb.EpochArray.interval(t_coast0_start.to(rep = 'NUM'),\n",
    "                                      t_burn0_start.to(rep = 'NUM')+dt, dt, sys = 'TDB')\n",
    "fm_coast0   = scb.ForceModelTranslation(primary_body=Orbiter, cannonball_SRP=True)\n",
    "prop_coast0 = scb.Propagator(\n",
    "    primary_body=Orbiter, state_vector=initial_state,\n",
    "    tspan=ep_coast0, force_models=fm_coast0,\n",
    ")\n",
    "\n",
    "# ── Finite burn arc 0 ────────────────────────────────────────────\n",
    "ep_burn0  = scb.EpochArray.interval(t_burn0_start.to(rep = 'NUM'), \n",
    "                                    t_burn0_end.to(rep = 'NUM'), \n",
    "                                    dt, sys = 'TDB')\n",
    "sv_burn0  = empty_state(Orbiter, origin, frame)\n",
    "fm_burn0  = scb.ForceModelTranslation(\n",
    "    primary_body=Orbiter, cannonball_SRP=True,\n",
    "    finite_burn=True, maneuver=burns[0],\n",
    ")\n",
    "prop_burn0 = scb.Propagator(\n",
    "    primary_body=Orbiter, state_vector=sv_burn0,\n",
    "    tspan=ep_burn0, force_models=fm_burn0,\n",
    ")\n",
    "\n",
    "# ── Final coast (6 hours after burn 0) ───────────────────────────\n",
    "coast_final_start_num = ep_burn0.times.values[-1]\n",
    "coast_final_end_num   = coast_final_start_num + 6 * 3600  # 6 hours in seconds\n",
    "ep_coast_final      = scb.EpochArray.interval(coast_final_start_num, \n",
    "                                              coast_final_end_num + dt_sec, dt, sys = 'TDB')\n",
    "sv_coast_final      = empty_state(Orbiter, origin, frame)\n",
    "fm_coast_final      = scb.ForceModelTranslation(primary_body=Orbiter, cannonball_SRP=True)\n",
    "prop_coast_final    = scb.Propagator(\n",
    "    primary_body=Orbiter, state_vector=sv_coast_final,\n",
    "    tspan=ep_coast_final, force_models=fm_coast_final,\n",
    ")\n",
    "\n",
    "print('Propagators built:')\n",
    "for label, ep in [('Coast before', ep_coast0), ('Burn 0', ep_burn0), ('Final coast', ep_coast_final)]:\n",

    "    t0, tf = ep.to(rep = 'CAL').times[0], ep.to(rep = 'CAL').times[-1]\n",

    "          f\"{t0}  →  {tf}\")\n",
    "\n",
    "print('~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~')\n",
    "print('Coast 0 last: ', ep_coast0.times.values[-1])\n",
    "print('Burn 0 first:', ep_burn0.times.values[0])\n",
    "print('Burn 0 last: ', ep_burn0.times.values[-1])\n",
    "print('Coast f first:', ep_coast_final.times.values[0])"

    "## 5. Assemble and Propagate the Mission Sequence\n",
    "\n",
    "`MissionSequence` chains the arcs in order.  The propagation engine:\n",
    "\n",
    "1. Propagates the **coast leg** from the given initial state.\n",
    "2. Inserts an automatic **node** at the burn start (no state discontinuity).\n",
    "3. Propagates **burn 0** forward, applying thrust and mass depletion at each step.\n",
    "4. Inserts another node at the burn end.\n",
    "5. Propagates the **final coast** from the post-burn state.\n",
    "\n",
    "Use `addLeg` for coast arcs and `addFiniteBurn` for thrust arcs."
   ]

    "## 7. Mass Evolution During Finite Burns\n",
    "\n",
    "During a finite-burn arc the Scarabaeus propagator integrates the mass ODE\n",
    "\n",
    "$$\\dot{m}(t) = -\\dot{m}_{burn}$$\n",
    "\n",
    "and stores the result as an `SCBPolynomial` inside the spacecraft.\n",
    "After propagation, `Spacecraft.mass(t)` evaluates this polynomial at any epoch\n",
    "within the valid domain of each arc.\n",
    "\n",
    "We call it at every epoch in `scenario.total_epochsTDB` to reconstruct the\n",
    "full mass history across coast and burn arcs."
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "id": "14",
   "metadata": {},
   "outputs": [],
   "source": [
    "# Evaluate mass at every propagated epoch\n",
    "# Orbiter._mass_profile is updated by the propagator for each arc;\n",
    "# mass(t) dispatches to the correct arc polynomial via get_mass().\n",
    "mass_vals = np.array(\n",
    "    [float(np.asarray(Orbiter.mass(float(t)).values).ravel()[0]) for t in t_all]\n",
    ")\n",
    "\n",
    "m0     = mass_vals[0]\n",
    "m_end  = mass_vals[-1]\n",
    "mdot   = burns[0].mass_flow.values[0]\n",
    "delta_m = m0 - m_end\n",
    "\n",
    "print(f'Initial mass    : {m0:.3f} kg')\n",
    "print(f'Final mass      : {m_end:.3f} kg')\n",
    "print(f'Propellant used : {delta_m:.3f} kg')\n",
    "print(f'Mass-flow rate  : {mdot:.4e} kg/s')\n",
    "\n",
    "# Plot only during burn, scatter style\n",
    "burn_mask = (hours >= h_burn_start) & (hours <= h_burn_end)\n",
    "\n",
    "h_plot = hours[burn_mask][:-1]\n",
    "m_plot = mass_vals[burn_mask][:-1]\n"

    "## 9. Notes and Common Pitfalls\n",
    "\n",
    "**Epoch continuity**: the `MissionSequence` validator checks that the last epoch of\n",
    "each arc matches the first epoch of the next.  `make_epochs` ensures `t_arr[-1] == tf`\n",
    "so there is no floating-point gap at arc boundaries.\n",
    "\n",
    "**NaN-initialized states**: non-first legs must be constructed with NaN\n",
    "position/velocity and a NaN epoch.  The sequence propagator fills these at\n",
    "runtime from the terminal state of the preceding arc.  Providing real values\n",
    "for subsequent arcs is silently ignored.\n",
    "\n",
    "**`addFiniteBurn` vs `addLeg`**: use `addFiniteBurn` for any arc whose\n",
    "`ForceModelTranslation` was built with `finite_burn=True`.  Mixing them\n",
    "(e.g., `addLeg` with a thrust force model) will propagate without thrust.\n",
    "\n",
    "**Nodes are inserted automatically**: `addFiniteBurn` inserts a node before\n",
    "the burn, and `addLeg` following a burn inserts one before the coast.\n",
    "Avoid calling `addNode` manually unless you need a named boundary marker.\n",
    "\n",
    "**Mass units**: `ManeuverParser` stores `mass_flow` in `kg/sec` and `thrust` in `N`.\n",
    "The Scarabaeus propagator stores thrust internally in `kN` so that dividing by\n",
    "mass in kg yields acceleration directly in `km/s²`."
