Scarabaeus Tutorials
====================
The tutorials section is broken up into three levels of complexity:

- **Basics**: illustrate the building blocks and basic functionalities of Scarabaeus
- **Intermediate**: demonstrate how to combine components from basics tutorials and use them to create more complex Scarabaeus objects
- **Advanced**: walk through full analysis scenarios from beginning to end, employing concepts learned from basics and intermediate tutorials

It is advised to begin with the Basics before moving on to Intermediate and finally Advanced. See below for the recommended order for each 
tutorial level.

Additionally, the file `RunAllDemos.py` is used to check functionality across all tutorials.

Basics
------
Recommended order to go through Basics tutorials:

1. `basics_Units.py`
2. `basics_EpochArray.py`
3. `basics_Frame.py`
4. `basics_CelestialBody.py`

Optionally, once completing the above, in any order:
- `basics_SpiceManager.py`
- `basics_Constants.py`
- `basics_OrbitalElements.py`
- `basics_FiniteBurnList.py`
