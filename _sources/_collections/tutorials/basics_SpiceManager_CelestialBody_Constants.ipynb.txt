    "### Body State Queries\n",
    "\n",
    "For any body in the kernel pool, `SpiceManager` returns its position, velocity, or full\n",
    "state relative to an observer in a chosen reference frame.\n",
    "\n",
    "It contains a few different methods that return different information depending on what you need:\n",
    "\n",
    "- `get_pos(target, et, frame, observer)`      : 3-vector position (km)\n",
    "- `get_vel(target, et, frame, observer)`      : 3-vector velocity (km/s)\n",
    "- `get_state(target, et, frame, observer)`    : 6-vector [pos; vel]\n",
    "- `get_lighttime(target, et, frame, observer)`: one-way light-time (s)\n",
    "\n",
    "We can see all of these values by getting the state of the Earth with respect to the Sun in J2000:"
   ]
  },

    "print(f\"Position only:\\n{scb.SpiceManager.get_pos('Earth', et, J2000.name, 'Sun')}\\n\")\n",
    "print(f\"Velocity only:\\n{scb.SpiceManager.get_vel('Earth', et, J2000.name, 'Sun')}\\n\")\n",
    "print(f\"Full state vector:\\n{scb.SpiceManager.get_state('Earth', et, J2000.name, 'Sun')}\\n\")\n",
    "print(f\"Light time:\\n{scb.SpiceManager.get_lighttime('Earth', et, J2000.name, 'Sun')}\\n\")"

    "### Querying Ephemerides\n",
    "\n",
    "If the body exists in the kernel pool, `get_state` returns its position and velocity at any epoch.\n",
    "Below we query all eight planets' heliocentric orbits over two years and plot them."
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "id": "15",
   "metadata": {},
   "outputs": [],
   "source": [
    "# 2 year time interval with weekly sample rate\n",
    "epochs = scb.EpochArray.interval(start = '2001 JAN 01 12:00:00.000', \n",
    "                                 end   = '2003 JAN 01 12:00:00.000',\n",
    "                                 dt    = scb.ArrayWUnits(7, day),\n",
    "                                 sys   = 'TDB')\n",
    "\n",
    "# query heliocentric state for each planet\n",
    "planets = ['MERCURY', 'VENUS', 'EARTH', 'MARS', \n",
    "           'JUPITER', 'SATURN', 'URANUS', 'NEPTUNE']\n",
    "planet_traj  = {}\n",
    "for planet in planets:\n",
    "    cb     = scb.CelestialBody.from_constants(planet)\n",
    "    states = []\n",
    "    print(f'Querying {planet}...')\n",
    "    for epoch in epochs:\n",
    "        state = cb.get_state(epoch_0 = epoch, reference_frame = 'J2000',\n",
    "                             origin = 'SUN').quantity\n",
    "        states.append(state[0:2].convert_to(AU).values)\n",
    "    planet_traj[planet] = np.array(states)\n",
    "    print(f'Retrieved all states across {epochs.duration(day)} interval')\n",
    "\n",
    "supp.supp_plotting.plot_planets(planets, planet_traj, epochs)"
   ]
