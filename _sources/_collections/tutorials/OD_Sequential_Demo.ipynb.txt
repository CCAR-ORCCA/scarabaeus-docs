    "## 2. LKF with SNC Process Noise\n",
    "\n",
    "### Theory\n",
    "The **Linearized Kalman Filter (LKF)** is the core sequential estimator.\n",
    "At each measurement epoch $t_k$:\n",
    "\n",
    "1. **Time update (predict):** propagate state and covariance forward with the STM\n",
    "   $$\\mathbf{P}_{k|k-1} = \\mathbf{\\Phi}_{k,k-1}\\,\\mathbf{P}_{k-1|k-1}\\,\\mathbf{\\Phi}_{k,k-1}^T + \\mathbf{Q}_k$$\n",
    "\n",
    "2. **Measurement update (correct):** incorporate new measurement $y_k$\n",
    "   $$\\mathbf{K}_k = \\mathbf{P}_{k|k-1}\\,\\mathbf{H}_k^T\\,(\\mathbf{H}_k\\mathbf{P}_{k|k-1}\\mathbf{H}_k^T + \\mathbf{R}_k)^{-1}$$\n",
    "   $$\\hat{\\mathbf{x}}_{k|k} = \\hat{\\mathbf{x}}_{k|k-1} + \\mathbf{K}_k(y_k - \\mathbf{H}_k\\hat{\\mathbf{x}}_{k|k-1})$$\n",
    "\n",
    "### State Noise Compensation (SNC)\n",
    "SNC models uncharacterised accelerations as **white noise** added continuously to the velocity:\n",
    "\n",
    "$$\\mathbf{Q}_k = \\int_{t_{k-1}}^{t_k} \\mathbf{\\Phi}(t_k,\\tau)\\,\\mathbf{B}\\,q_c\\,\\mathbf{B}^T\\,\\mathbf{\\Phi}(t_k,\\tau)^T\\,d\\tau$$\n",
    "\n",
    "where $q_c$ [km²/s³] is the **continuous-time noise spectral density**.\n",
    "\n",
    "**Tuning Q_cont (SNC):**\n",
    "- Too large → wide covariance, accepts noise, position sigma grows fast.\n",
    "- Too small → filter stiffens, can't track maneuvers or unmodelled forces.\n",
    "- Rule of thumb: $q_c \\approx (\\text{expected unmodelled accel})^2 / \\text{arc length in sec}$\n",
    "\n",
    "```python\n",
    "Q_cont = scb.ProcessNoiseSettings(type='SNC', Q_cont=Q_cont_matrix)\n",
    "```\n"
   ]

    "## 3. Process Noise Deep Dive: SNC vs. DMC\n",
    "\n",
    "### State Noise Compensation (SNC)\n",
    "- Models uncharacterised forces as **continuous white noise on velocity**.\n",
    "- Does **not** increase state dimension.\n",
    "- `Q_cont` is a 3×3 matrix [km²/s⁴] — one spectral density per acceleration axis.\n",
    "- Typical values: 1e-18 (well-modelled) to 1e-12 (significant unmodelled forces) km²/s⁴.\n",
    "- Increases covariance between measurements, preventing filter overconfidence.\n",
    "\n",
    "### Dynamical Model Compensation (DMC)\n",
    "- Models uncharacterised forces as a **first-order Gauss-Markov (FOGM)** stochastic process.\n",
    "- The FOGM acceleration `a_fogm` is **part of the state** and is propagated.\n",
    "- Requires `ForceModelTranslation(first_order_gauss_markov=True, fogm_beta=...)`\n",
    "- Process noise type: `'DMC'` with `beta=force_model.fogm_beta`.\n",
    "\n",
    "```python\n",
    "# SNC — no state augmentation\n",
    "pn = scb.ProcessNoiseSettings(type='SNC', Q_cont=Q_matrix)\n",
    "\n",
    "# DMC — augmented state with FOGM acceleration\n",
    "pn = scb.ProcessNoiseSettings(type='DMC', Q_cont=Q_matrix,\n",
    "                               beta=force_model.fogm_beta)\n",
    "```\n",
    "\n",
    "### When to use which\n",
    "| Scenario | Recommendation |\n",
    "|----------|----------------|\n",
    "| Short arc, well-modelled dynamics | SNC (small Q) |\n",
    "| Long arc with slow-varying unmodelled forces | DMC (captures temporal correlation) |\n",
    "| Maneuver estimation during arc | DMC + FOGM |\n",
    "| Batch filter | Neither (use PFOGM instead) |\n",
    "\n",
    "### Correlation time and β\n",
    "In the FOGM model the force autocorrelation decays as $e^{-\\beta \\Delta t}$.\n",
    "- Large β (e.g. 1/60 s⁻¹) → force decorrelates in ~1 minute (rapid white-noise-like behaviour)\n",
    "- Small β (e.g. 1/86400 s⁻¹) → force has memory over ~1 day (slow trend)\n",
    "\n",
    "Rule of thumb: set β so the correlation time 1/β matches the expected timescale of the\n",
    "unmodelled perturbation.\n"
   ]

    "## 5. SRIF Sequential Filter\n",
    "\n",
    "The **Square-Root Information Filter (SRIF)** is the numerically most stable sequential algorithm.\n",
    "Instead of propagating the covariance **P**, it propagates the square root of the information\n",
    "matrix **R = P⁻¹**, avoiding the numerical issues associated with covariance inversion.\n",
    "\n",
    "SRIF is preferred when:\n",
    "- The state dimension is large (many parameters)\n",
    "- Long arcs with many measurements accumulate numerical error in P\n",
    "- Near-singular covariance matrices arise (highly correlated parameters)\n",
    "\n",
    "API is identical to LKF — just swap `scb.LKF` for `scb.SRIF`.\n"

    "## 6. RTS Smoother\n",
    "\n",
    "The **Rauch-Tung-Striebel (RTS) smoother** runs a backward pass after the forward Kalman filter\n",
    "to refine estimates at *earlier* epochs using measurements from *later* epochs.\n",
    "\n",
    "- The forward filter is causal: at $t_k$ it uses only measurements $t \\leq t_k$.\n",
    "- The smoother is acausal: at $t_k$ it uses **all** measurements in the arc.\n",
    "\n",
    "Enable it by passing `if_sequential_smooth=True` to `filter.fit()`.\n",
    "The smoothed solution is stored in `solution.deviation_smooth` and `solution.covariance_smooth`.\n",
    "\n",
    "**When to use:** post-processing after all data are collected.\n",
    "**Not applicable for:** real-time navigation (forward-only).\n"
   ]

    "## 8. Consider Parameters\n",
    "\n",
    "A **consider parameter** is a physical quantity that:\n",
    "1. Has significant uncertainty and affects the measurements (it is *observable*).\n",
    "2. Is **not** solved for — it is held at its reference value during the solve.\n",
    "3. Its uncertainty is carried forward into the state covariance (conservative covariance inflation).\n",
    "\n",
    "This is useful when a parameter cannot be resolved from the available data but ignoring its\n",
    "uncertainty would produce an over-optimistic covariance.\n",
    "\n",
    "```python\n",
    "# Estimated: filter updates this parameter's value\n",
    ".param('eta_srp', sc, value, estimation='estimated', dynamics='static')\n",
    "\n",
    "# Considered: uncertainty propagated but value not updated\n",
    ".param('eta_srp', sc, value, estimation='considered', dynamics='static')\n",
    "```\n",
    "\n",
    "Typical consider parameters in OD:\n",
    "- Ground station position error\n",
    "- Transponder delay calibration\n",
    "- SRP scale factor (when observable evidence is weak)\n",
    "- Range bias\n"
   ]
