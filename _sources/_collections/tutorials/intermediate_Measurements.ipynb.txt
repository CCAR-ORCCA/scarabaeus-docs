    "## 7. Partials and Residuals\n",
    "\n",
    "### Partials (H matrix)\n",
    "\n",
    "`compute_partials(target, epoch_array, frame)` returns the measurement partial-derivative\n",
    "matrix $H_k = \\partial h / \\partial x$ evaluated at each epoch.  This is the same matrix\n",
    "the filter uses to map state corrections to measurement corrections.  It is normally\n",
    "computed internally by `generate_measurement_dataset`, but can be accessed directly here.\n",
    "\n",
    "### Residuals (O − C)\n",
    "\n",
    "`residuals(observed, computed)` returns the pre-fit residual vector\n",
    "$y_k = z_k^\\text{obs} - h(x^*(t_k))$ at each epoch."
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "id": "18",
   "metadata": {},
   "outputs": [],
   "source": [
    "# ── partials — compute_partials returns a list of lists → stack to ndarray\n",
    "H_range     = np.array(Range_GS1.compute_partials(    Orbiter_ref, ep_range_et,     J2000))\n",
    "H_rangerate = np.array(RangeRate_GS1.compute_partials(Orbiter_ref, ep_rangerate_et, J2000))\n",
    "H_opnav     = np.array(Centroid.compute_partials(     opnav_target, ep_opnav_et,    J2000))\n",
    "\n",
    "print(f\"H range      : {H_range.shape}      (n_meas × n_state)\")\n",
    "print(f\"H range rate : {H_rangerate.shape}      (n_meas × n_state)\")\n",
    "print(f\"H opnav      : {H_opnav.shape}   (2 × n_epochs × n_state)\")\n",
    "\n",
    "# ── residuals — residuals() returns ArrayWFrame; extract .quantity.values\n",
    "res_range     = Range_GS1.residuals(    meas_range,     comp_range    ).quantity.values.ravel()\n",

    "\n",
    "print(f\"\\nRMS range residual     : {float(np.sqrt(np.mean(res_range**2))):.4e} km\")\n",
