    "---\n",
    "## 1. N-Plate Model\n",
    "\n",
    "The **N-Plate SRP model** represents a spacecraft's solar-radiation-pressure geometry\n",
    "as a finite set of flat plates, each with its own area, reflectivity coefficients, and\n",
    "surface normal.  Panels can be **fixed** in the spacecraft body frame or **attitude-driven**\n",
    "by a SPICE C-kernel (CK), as is the case for articulated solar arrays.\n",
    "\n",
    "The model is constructed from a JSON configuration file that specifies each panel's\n",
    "physical properties and SPICE frame ID.  The two meta-kernel types loaded above are:\n",
    "\n",
    "| Kernel type | Purpose |\n",
    "|---|---|\n",
    "| CK (spacecraft body) | Main-bus attitude → body-frame orientation in J2000 |\n",
    "| CK (solar arrays) | Solar-panel articulation → time-varying panel normals |\n",
    "\n",
    "### Constructing the Model"
   ]
  },

    "---\n",
    "## 2. Spacecraft Attitude\n",
    "\n",
    "`Spacecraft.get_attitude(frame, et)` queries the C-kernel and returns the orientation\n",
    "of the spacecraft body frame as a unit quaternion $[q_0,\\, q_1,\\, q_2,\\, q_3]$ at the\n",
    "given epoch.  This wraps the underlying SPICE `ckgpav` / `m2q` calls and expresses\n",
    "the result in the requested inertial frame.\n",
    "\n",
    "The example below extracts the quaternion history for OSIRIS-REx over a five-day window\n",
    "and plots all four components."
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "id": "18",
   "metadata": {},
   "outputs": [],
   "source": [
    "# ── attitude query over a 5-day window ───────────────────────────\n",
    "et_start = scb.SpiceManager.str2et(\"2021 AUG 17 12:00:00 UTC\")\n",
    "et_end   = scb.SpiceManager.str2et(\"2021 AUG 22 08:00:00 UTC\")\n",
    "step_sec = 20                                          # sample every 20 s\n",
    "\n",
    "et_arr = np.arange(et_start, et_end, step_sec)\n",
    "\n",
    "# reuse the spacecraft defined in §1 (orex already has the N-plate model)\n",
    "attitudes = [orex.get_attitude(J2000, et) for et in et_arr]\n",
    "attitudes = np.array(attitudes)                        # shape: (N, 4)\n",
    "\n",
    "print(f\"Queried {len(attitudes)} attitude epochs\")\n",
    "print(f\"First quaternion  [q0 q1 q2 q3] : {attitudes[0].round(6)}\")\n",
    "print(f\"Norm check (should be 1.0)       : {np.linalg.norm(attitudes[0]):.8f}\")"
   ]
