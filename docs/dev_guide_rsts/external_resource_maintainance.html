Scarabaeus External Resource Guide#

Introduction#

Intro here

Besides the functionality itself, there are two other important components of writing a class for Scarabaeus:

1. Auto-generated documentation: providing information for display here in Scarabaeus’ resources
2. In-code documentation: general commenting for the actual code

The class TemplateClassV2 has been set up as an example to follow when writing a class or methods for Scarabaeus.

Note

The template code is available in its entirety within Scarabaeus under TemplateClassV2.py, or here.

See also

Generating Scarabaeus Documentation for information on building and modifying this resource.
