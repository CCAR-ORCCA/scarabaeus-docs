1. FilterDataManager#

class FilterDataManager#

Bases: object

A class responsible for managing and processing measurement data for use in orbit determination filters. The class provides methods to convert datasets to a standardized list format, combine multiple datasets, standardize times across measurements, and combine measurement time vectors into a unified time array.

The goal of this class is to facilitate the process of filtering and data assimilation by preparing the data for further analysis and model updates. The methods provided ensure that measurement data is consistently formatted, combined, and processed for use in filtering algorithms such as Kalman Filters or other estimation methods.

Notes

  • This class is designed to handle datasets from different sources, ensuring that the data is combined in a way that maintains chronological integrity.

  • It supports the integration of multiple measurement types (e.g., range, range rate) by combining their corresponding data vectors.

  • The times in different datasets may be in different time systems or units. Methods such as standardize_times are provided to ensure consistent time references across measurements.

  • The combine_multiple_datasets method allows you to merge datasets, ensuring that data is aligned based on timestamps.

Example

```python import scarabaeus as scb

# Initialize measurement models and trajectory models # …

# Generate range and range rate datasets range_dataset_GS1 = Range_GS1.generate_measurement_dataset(

“GS1 Range”, measurement_type=”range”, target=Orbiter_pert_it1, observed_meas=obs_quantities_range,

)

rangerate_dataset_GS1 = RangeRate_GS1.generate_measurement_dataset(

“GS1 Range Rate”, measurement_type=”rangerate”, target=Orbiter_pert_it1, observed_meas=obs_quantities_rangerate,

)

# Instantiate a Filter Data Manager object datasets_it1 = [range_dataset_GS1, rangerate_dataset_GS1]

measurement_data_it1, dataset_names_it1 = scb.FilterDataManager.combine_multiple_datasets(datasets_it1) ```

classmethod combine_multiple_datasets(datasets: list)#

Combines multiple MeasurementDataSets into a single dataset, sorted by timestamps.

This method combines data from multiple MeasurementDataSets into a list format that can be ingested by filters. It ensures that measurements are ordered by timestamps and handles datasets with different lengths.

Parameters:

datasets (list) – A list of MeasurementDataSets to be combined.

Returns:

  • combined_data: A list of combined measurement data, sorted by timestamp.

  • dataset_names: A list of the dataset names.

Return type:

tuple

Raises:

Warning – If no datasets or an invalid input is provided, a warning is raised and None is returned.

classmethod convert_dataset_to_list(dataset: MeasurementDataSet) → list#

Converts a MeasurementDataSet object into a list format with unitless data for filter processing.

The data is output in the form of: [[time_1, observed_1, computed_1, residuals_1, partials_1, covariance_1, instrument],

… [time_n, observed_n, computed_n, residuals_n, partials_n, covariance_n, instrument]]

Parameters:

dataset (scb.MeasurementDataSet) – The dataset containing measurement data to be converted.

Returns:

A list of measurement data, with each element containing the time, observed value, computed value, residuals, partial derivatives, covariance, instrument, measurement type, and dataset name.

Return type:

list

standardize_times(dataset, target_timeframe)#

Standardizes the times of a dataset to a specific time system or timeframe.

This method adjusts the times in the dataset to match a target time system, ensuring consistency across all measurements for further processing.

Parameters:

  • dataset – The dataset whose times need to be standardized.

  • target_timeframe – The target time system to which the dataset times should be converted.

Returns:

None
