5. OutlierRejection#

class OutlierRejection(ax: Axes, x: ndarray, y: ndarray, marker: str = '.', label: str = '')#

Bases: object

Handles the selection and removal of the outliers points from the plots.

Allows the users to select a set of points using the lasso selector and remove them.

Attributes

ax

Axes object of the plot

canvas

Canvas attribute of the plot.

label

Label for the plot.

lasso

Lasso object to handle the mouse events.

marker

Marker of the plot.

removed_indices

List to save the indices of the outlier points

scatter

Scatter plot object.

x

X-coordinates of the plot.

y

Y-coordinates of the plot.

static apply_lasso(axs: Axes, data: list)#

Applies the lasso selector to multiple axes and returns the updated data.

param data: axs type data: list of matplotlib.axes.Axes param data : data - (args to create a new outlier rejection object) type data: list of tuples return: removed_indices rtype : list of lists

Example

>>> fig, axs = plt.subplots(1, 2)
>>> x1, y1 = [1,24,5,6],[3,5,6,5]
>>> x2, y2 = [2,4,52,1],[7,6,6,8]
>>> data = [(x1, y1, '.', 'GS1'), (x2, y2, '.', 'GS1')]
>>> removed_indices_list = LassoHandler.apply_lasso(axs, data)
Parameters:

  • data (list)
